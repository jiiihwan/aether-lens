# 🔺 AetherLens (에테르렌즈)

**AetherLens**는 100% 웹 브라우저 로컬 연산(OpenCV.js WASM)과 구글 Gemini Flash API를 결합한 하이브리드 **AI 여행 사진 선별 및 자동 보정 에이전트**입니다. 

수많은 여행 사진 중 초점이 흐리거나 구도가 겹치는 연사를 걸러내고, 미적으로 우수한 베스트 컷을 골라 감성적인 한글 여행 앨범북과 자동 보정본을 일괄 제작해 줍니다.

---

## 🌟 핵심 기능 소개

### 1. 사진 대기열 제어 및 관리 (Staging Area)
* 파일을 드래그 앤 드롭하거나 탐색기 버튼으로 추가하여 정리 대기열을 구성합니다.
* 에이전트를 가동하기 전에 대기열에서 개별 사진을 자유롭게 미리보고 삭제할 수 있는 안전 진입형 구조입니다.

### 2. 정밀 흔들림 감지 (OpenCV.js WASM)
* 웹브라우저 상에서 로컬 WASM으로 구동되는 OpenCV.js를 활용하여 각 사진의 라플라시안 선명도(Laplacian Variance) 분산값을 계산합니다.
* 사용자가 설정한 선명도 임계값 기준 미달인 초점 나간 흐린 사진을 자동으로 탈락시킵니다.

### 3. 미적 품질 개별 채점 (Aesthetic Scorer)
* 로컬 화질 통계를 기반으로 개별 사진의 미적 점수를 **100점 만점** 기준으로 공정하게 산정합니다.
* 사진의 밝기 분포도(조도 밸런스), 명암 대비 표준편차, 선명도 가점을 합산하여 미적 퀄리티를 상세히 계량화합니다.

### 4. 중복 및 연사 제거 (Gemini Vision 구도 오디션)
* pHash(Perceptual Hash) 기반 해싱을 통해 구도가 유사한 연사 그룹을 시간대별로 묶어냅니다.
* 중복 그룹 중 가장 미적 점수가 높은 2장의 사진을 후보로 선정하여, **Gemini Vision API**에 예술적 구도 심사를 위뢰합니다.
* 수평 안정감, 인물의 미소, 구도의 생동감을 종합 평가하여 최종 단 1장의 대표 베스트 컷을 엄격하게 선정합니다.

### 5. 자동 이미지 보정 (CLAHE)
* 최종 엄선된 베스트 사진들에 대해 OpenCV의 **CLAHE(Contrast Limited Adaptive Histogram Equalization)** 명암 조절 알고리즘을 로컬 연산 적용합니다.
* 역광이나 그늘진 톤을 미려하고 선명하게 자동 톤 보정합니다.

### 6. 여행 스토리라인 에피소드 앨범북 구축
* 촬영 시각의 공백을 감지하여 여정을 시간차 에피소드로 자동 분류(Clustering)합니다.
* Gemini AI를 호출하여 전체 여행을 관통하는 대제목과 서평, 그리고 각 챕터별 한글 감성 에피소드 스토리와 사진 매칭용 캡션을 매핑해 줍니다.
* 앨범 이미지 확대/축소 슬라이딩 버튼을 통해 **2열, 3열, 5열로 유연한 핀터레스트 스타일 가로 등분 렌더링**을 제어할 수 있습니다.

## ⚡ 최신 성능 개선 사항

* **실시간 가로 프로그레스 바 장착**:
  * 에이전트 가동 상태 터미널 내부에 백분율 진행도와 유동적으로 밀착되는 미려한 보라-시안 그라데이션 광채 진행률 바를 탑재하여 시각적 진척도를 개선했습니다.
* **강력한 API 예외 차단 및 우회 복원(Robustness)**:
  * **사전 Key 검증**: 시작 버튼 클릭 시 API Key 누락 및 오타를 선제 검사하여 확인 창을 노출합니다.
  * **2중 우회 confirm 인터랙션**: API Key가 아예 없거나 구동 중 구글 사용량 초과(429 Rate Limit) 오류 발생 시, 팝업 질문을 통해 브라우저 **로컬 전용 WASM 화질 분석 파이프라인으로 1초 만에 즉각 자동 전환(Fallback)**하여 중단 없는 완수를 보장합니다.
  * **대기열 자동 복구(Rollback)**: 사용자가 API 제한 팝업에서 취소를 누르면, 가동 창을 숨기고 이전에 올려둔 대기열 사진 목록을 고스란히 복구하여 원상태로 돌려놓습니다.
* **Flexbox 등분 정렬 레이아웃**:
  * 기존 다단 칼럼 정렬의 사진 쏠림 렌더링을 완전히 제거하고, **CSS Flexbox 가로 비율 분할 수식**을 적용하여 사진 개수가 4장 이하로 적을 때에도 찌그러짐 없이 단정하고 예쁘게 좌측 정렬되도록 보정했습니다.
